### 開発方法

1. レポジトリ（[sshihci/sshihci.github.io](https://github.com/sshihci/sshihci.github.io)）からクローンする
   ```sh
   git clone https://github.com/sshihci/sshihci.github.io.git
   ```
2. クローンしたディレクトリを開く
   ```sh
   cd sshihci.github.io
   ```
3. 依存関係の解決
   ```sh
   yarn
   ```
4. 開発サーバーを立ち上げる
   ```sh
   yarn dev
   ```
5. http://localhost:8000 をブラウザで開く
